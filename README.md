# prompt-engineering-for-data-analytics

## Overview
This project demonstrates prompt engineering techniques for data analytics using Gemini.
The focus is on how prompt structure, constraints, and context injection affect insight quality.

## Key Skills Demonstrated
- Prompt Engineering
- Context Injection
- Hallucination Control
- Business Insight Framing
- GenAI Evaluation

## Tools
- Python
- Gemini API
- Jupyter Notebook
